
# Memecoin Trading Platform

A modern trading platform for memecoins built with Go and React/TypeScript.

## Features

- Real-time price tracking
- Portfolio management
- Trading interface
- Leaderboard system
- Support ticket system
- User authentication
- Wallet integration

## Tech Stack

### Backend
- Go 1.21+
- Gin (Web Framework)
- GORM (ORM)
- Redis (Caching)
- PostgreSQL (Database)
- JWT (Authentication)

### Frontend
- React 18
- TypeScript
- Redux Toolkit
- TailwindCSS
- Chart.js
- Vite

## Prerequisites

- Go 1.21+
- Node.js 18+
- Docker and Docker Compose
- PostgreSQL 14+
- Redis 6+
